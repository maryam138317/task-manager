# Task Manager App

A modern and responsive task management application built with **Next.js**, **SQLite**, and **Material UI**. The application helps users organize, prioritize, track, and analyze their daily tasks through an intuitive and user-friendly interface.

---

## 🚀 Features

### 📋 Task Management

* Create new tasks
* View all tasks
* Edit task details:

  * Title
  * Priority
  * Status
* Delete tasks

### 📅 Today's Tasks

* Dedicated page for displaying tasks scheduled for the current day
* Automatically filters tasks based on the current date

### ⚡ Priority Management

Tasks are categorized into three priority levels:

* Low Priority (Not Important)
* Important
* Urgent

Dedicated priority page for quick task organization and tracking.

### 📊 Statistics Dashboard

Visual insights into task progress using:

* Pie Chart
* Line Chart

Track productivity and monitor task completion trends.

### 🎨 Theme Customization

* Light Mode
* Dark Mode

Theme preferences are managed globally using React Context API.

### ⚙️ Settings Page

* Application overview
* Usage information
* Configuration options

## ⚡ Getting Started

### Prerequisites

* Node.js
* npm or yarn

### Installation

Clone the repository:

```bash
git clone https://github.com/maryam138317/task-manager.git
```

Navigate to the project folder:

```bash
cd task-manager
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```
